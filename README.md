# CalculatorGPT 🧮

A straightforward calculator application built with Python 3. Perform basic arithmetic operations with a simple, intuitive terminal interface.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🛡️ Error handling for invalid inputs
- 🔄 Continuous operation mode

## Requirements

- Python 3.x

## Installation

```bash
git clone https://github.com/Python-Swift15/CalculatorGPT.git
cd CalculatorGPT
```

## Usage

```bash
python3 calc.py
```

### How to Use

1. Run the program
2. Select an operation (1-5):
   - **1** = Add
   - **2** = Subtract
   - **3** = Multiply
   - **4** = Divide
   - **5** = Quit
3. Enter two numbers when prompted
4. View the result and repeat or exit

## Example

```
Welcome to CalculatorGPT
1) Add
2) Subtract
3) Multiply
4) Divide
5) Quit

Choose an operation: 1
Enter the first number: 15
Enter the second number: 7
15+7=22
```

## Technical Details

- Uses try-except blocks for robust error handling
- Validates user input before calculations
- Supports both integer and floating-point operations
- Loop-based design for multiple calculations
